# 📦 Logistics Delivery Analysis Dashboard

An interactive Power BI dashboard designed to analyze logistics and delivery performance across multiple dimensions.

---

## 🚀 Project Overview

This project was developed to help a logistics company understand and optimize its product delivery process using key performance indicators (KPIs) and data visualization.

---

## 📊 Dashboard Preview

![Dashboard](./images/dashboard-preview.png)

---

## 🔍 Key KPIs

- Total On-Time Deliveries by Delivery Channel  
- Percentage of Early Deliveries by Delivery Team  
- Total Deliveries Per Month  
- Total Deliveries from Top 5 Sellers  
- Total Delayed Deliveries Per City  
- Delivery Status Distribution (%)  

---

## 📈 Key Insights

- Majority of deliveries are completed on time  
- Early deliveries represent a strong performance indicator  
- Certain teams outperform others in early deliveries  
- Seasonal variations impact delivery volume  
- Delays are concentrated in specific cities
