# 📦 Logistics Delivery Analysis Dashboard

An interactive Power BI dashboard designed to analyze logistics and delivery performance.

---

## 🚀 Project Overview

This project helps a logistics company understand delivery performance using key KPIs and data visualization.

---

## 📊 Dashboard Preview

![Dashboard](https://github.com/Resaletti/logistics-delivery-analysis-dashboard/blob/80e47ceec2ed6a4c2b4afaa08ad21b0040743108/images/Logistics.png)

---

## 🔍 Key KPIs

- On-Time Deliveries by Channel  
- Early Deliveries by Team  
- Deliveries per Month  
- Top 5 Sellers  
- Delayed Deliveries per City  
- Delivery Status (%)  

---

## 🛠 Tools

Power BI | DAX | Data Modeling
